# Mogura
もぐらたたきゲーム

## ルール
- 敵をクリックで倒す
- スペシャル敵を倒すと一定時間触れただけで敵が倒せる
- 触れてはいけない敵を倒すと減点(スペシャル状態のときは普通に倒せる)
